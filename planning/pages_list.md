# Pages List

This document outlines all the pages included in our web application. The pages list are as follows:

---

## 1. Register Page
- Description: Allows new users to sign up.
- Features:
  - Students use their student email ID
  - Create own password

---

## 2. Login Page
- Description: Allows approved students to log into the platform.
- Features:
  - University email login
  - Input validation and error handling

---

## 3. Home Page
- Description: Main page after login displaying all user-generated content.
- Features:
  - Text and image posts
  - Lost & Found posts
  - Marketplace listings
  - Club posts and announcements
  
---

## 4. Lost & Found Page
- Description: Dedicated section for lost and found items.
- Features:
  - Posts about lost or found items
  - Image uploads
  - Comment section
  - Option to message user

---

## 5. Marketplace Page
- Description: Allows students to buy and sell used items.
- Features:
  - Item listings (e.g., electronics, books)
  - Post creation for selling items
  - Images, price, and basic details shown directly in feed

---

## 6. Clubs & Societies Page
- Description: Displays university clubs and societies upcoming events.
- Features:
  - List of clubs
  - Navigation to individual club pages
  - Club event details (like location, date and time)

---

## 7. Post Creation Page
- Description: Allows users to create new posts.
- Features:
  - Select post type (Lost & Found, Marketplace, Club, General)
  - Add text and images
  - Submit post

---

## 8. Profile Page
- Description: Displays user profile information.
- Features:
  - User details
  - User’s posts and activity
