# User Stories 1

1. As a UWA student, I want to sign up and log in with my university email, so that only approved students can access the platform.

2. As a user, I want to create text or image posts, so that I can share information, ask questions, or make announcements in the student community.

3. As a user, I want to create a lost item post, so that other students can help me find something I have lost on campus.

4. As a user, I want to create a found item post, so that I can help return a lost belonging to its owner.

5. As a user, I want to comment on posts, so that I can ask for more details, give helpful information, or interact with other students.
