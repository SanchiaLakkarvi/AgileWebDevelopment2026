# User Stories 2
1. As a user, I want to like or dislike posts, so that the most useful or relevant content becomes more visible to others.
2. As a user, I want to browse posts by category (e.g., Lost & Found, Marketplace, Clubs), so that I can quickly find relevant content.
3. As a user, I want to create a listing in the marketplace, so that I can sell or give away used items to other students.
4. As a user, I want to edit or delete my posts, so that I can correct mistakes or remove outdated information.
5. As a user, I want to view club or society pages, so that I can stay updated on events, activities, and announcements.
6. As a club representative, I want to post updates on my club page, so that I can inform students about upcoming events and opportunities.
7. As an admin, I want to approve or restrict user access based on UWA email verification, so that only authorized students can access the platform.
8. As an admin, I want to remove inappropriate posts or comments, so that the platform remains safe and respectful for all users.
9. As a user, I want to search for posts using keywords, so that I can quickly find specific information.
10. As a user, I want to mark an item as sold, so that others know it is no longer available.
